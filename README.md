# EDA_on_titanic_dataset
This project performs an in-depth Exploratory Data Analysis (EDA) on the famous Titanic dataset, aiming to uncover patterns, trends, and relationships that influenced passenger survival.
🚢 Titanic Dataset - Exploratory Data Analysis (EDA)
📌 Overview

This project focuses on performing Exploratory Data Analysis (EDA) on the Titanic dataset to uncover patterns and factors that influenced passenger survival.

The analysis includes data cleaning, visualization, and statistical exploration to better understand the dataset and prepare it for machine learning models.

🎯 Objectives
Understand the structure and features of the dataset
Handle missing and inconsistent data
Analyze relationships between different variables
Identify key factors affecting survival
📊 Dataset Information

The dataset contains information about Titanic passengers, including:

PassengerId
Name
Sex
Age
Pclass (Passenger Class)
Fare
Embarked
Survival status
🛠️ Tools & Technologies
Python 🐍
Pandas
NumPy
Matplotlib
Seaborn
Jupyter Notebook
🔍 EDA Process
1. Data Cleaning
Handled missing values (Age, Embarked, Cabin)
Converted data types where necessary
Removed or imputed null values
2. Univariate Analysis
Distribution of Age, Fare
Count plots for categorical variables (Sex, Pclass, Embarked)
3. Bivariate Analysis
Survival rate by Gender
Survival rate by Passenger Class
Age vs Survival
Fare vs Survival
4. Multivariate Analysis
Combined analysis of multiple features
Heatmaps for correlation
📈 Key Insights
👩 Females had a much higher survival rate than males
🎟️ First-class passengers were more likely to survive
👶 Younger passengers showed slightly better survival trends
💰 Higher fares were associated with higher survival probability
